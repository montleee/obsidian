`ArrayList<E>` — коллекция на основе массива
`LinkedList<E>`— коллекция на основе _двусвязного списка_ состоит из нод которые хранят значения и ссылку на предыдущий и следующий элемент
#### Доступ по индексу
`ArrayList<E>` - O(1)
`LinkedList<E>`- O(n)
#### Добавление в конец
`ArrayList<E>` - O(1)
`LinkedList<E>`- O(1)
#### Добавление/удаление  в середине 
`ArrayList<E>` - O(n)
`LinkedList<E>`- O(n) нужно найти нод
