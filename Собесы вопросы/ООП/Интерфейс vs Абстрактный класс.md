#### Определение 
`Интерфейс` - _контракт_, который определяет набор методов без реализации, а классы  реализуют его методы.

`Абстрактный класс` —  _Общий базовый класс_ с частичной реализацией может иметь абстрактные методы.
#### Множественное наследование 
`Интерфейс` - Класс может _реализовать несколько интерфейсов_. 
`Абстрактный класс` -  Класс может наследовать только _один абстрактный класс_.

#### Методы
`Интерфейс` -  
-  абстрактные методы  [ до Java 8 ] 
-  default методы и статические методы  [ Java 8 ] 
-  приватные методы. [ Java 9 ] 

`Абстрактный класс` - Может содержать как _абстрактные методы_, так и _методы с реализацией_.

#### Поля
`Интерфейс` - Поля в интерфейсе могут быть только _public static final_ (константы)

`Абстрактный класс` -  любые модификаторы доступа [private  protected  public].
